Part-Time Remote Working Student

Full-Stack Developer

Remote ~20 hours/week Contract

About the Role

Generative AI Teknoloji A.Ş. is an innovative startup building a web application powered by generative AI. We're working on cutting-edge computer vision and AI tools and turning them into intuitive, useful web products. As a working student on our team, you'll help make advanced AI capabilities accessible through a friendly web interface.

This is a hands-on role: you'll design features, integrate APIs, and see your code in production – all while gaining experience in a rapidly growing field.

We value curiosity and a "learn-by-doing" attitude. You'll join a small, agile team where learning new technologies is expected. English is our working language, and communication skills are important.

Responsibilities

Build and improve the web UI: Implement and refine pages and components of our web application, focusing on usability, responsiveness, and performance.

Work on AI-driven features: Help build flows where users can interact with AI-powered tools (e.g., content generation, personalization, smart templates).

Integrate external services and APIs: Connect the web app to third-party APIs (including AI services) and ensure data is handled securely and efficiently.

User management and data handling: Develop and maintain features like user authentication, profiles/dashboards, and secure storage of user-related data.

Feature expansion and quality: Work with the team to define and implement new features, test functionality, fix bugs, and contribute to a smooth user experience.

Learning and research: Stay up-to-date on modern web technologies and relevant AI tools. A key part of this job is being eager to learn new libraries, frameworks, and services as the product evolves.

Qualifications

Requirements

Student status: Currently enrolled in a Bachelor's or Master's program (Computer Science, Software Engineering, or a related field). This is a working-student (part-time) role.

Programming skills: Experience with web development. You should know HTML/CSS/JavaScript and at least one backend language (e.g., Python, Node.js). Familiarity with modern web frameworks (such as React or Vue) and REST APIs is a plus.

Interest in AI: Genuine enthusiasm for AI and modern web products. You don't need to be an ML expert, but you should be excited to work on AI-driven applications.

Team player with solid English: Good communication and collaboration skills. All work (code, documentation, discussions) is in English, so strong written and spoken English is required.

Self-motivated learner: We're looking for someone curious and proactive. You should be able to pick up new tools quickly, ask questions when needed, and work with minimal supervision.

Nice-to-Haves

Portfolio / projects involving web development, AI, or multimedia (GitHub, personal website, school projects, etc.)

Working Conditions & Compensation

Hours: Part-time (~20 hours/week). Flexible schedule (e.g., Monday, Wednesday, Friday), arranged around your classes.

Remote: 100% remote work – you can work from anywhere with a reliable internet connection.

Contract: Freelance/contract position (initial 6–12 months, extendable based on progress and mutual fit).

Salary: 40,000 TL net per month (Turkish Lira), paid monthly.

Tools & Support: You'll have access to the necessary tools and APIs, plus guidance from more experienced developers. We encourage mentorship, code reviews, and pair programming as part of your learning.

Why Join Us?

Help build a product from the ground up at the intersection of web development and AI.

Work in a collaborative environment with space to propose ideas, experiment, and grow your skills.

Gain hands-on experience contributing to a real-world product used by end users.

If you're passionate about learning and want to help shape the future of AI-powered web applications, we'd love to hear from you.

How to Apply

Send an email to hello@aistudio.works with the subject line: "Working Student – Full-Stack Developer"

Please include:

Your CV and a short introduction (in your preferred language, even 2–3 sentences is enough) explaining why you're interested in this role.

Links to any relevant projects, portfolios, or past work (if available).

Your expected salary for this position.
